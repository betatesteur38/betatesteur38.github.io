Résumé de JavaScript

1. Introduction

JavaScript est un langage de programmation utilisé pour rendre les pages web interactives. Il s'exécute côté client (dans le navigateur).

2. Syntaxe de base

JavaScript peut être intégré directement dans une page HTML avec la balise <script>.

<script>
    console.log("Bonjour, monde!");
</script>

3. Variables

Les variables peuvent être déclarées avec let, const ou var.

let nom = "Alice";
const age = 25;
var ville = "Paris";
console.log(`Je m'appelle ${nom}, j'ai ${age} ans et j'habite à ${ville}.`);

4. Structures de contrôle

Conditions

let heure = 14;
if (heure < 12) {
    console.log("Bonjour!");
} else if (heure < 18) {
    console.log("Bon après-midi!");
} else {
    console.log("Bonsoir!");
}

Boucles

// Boucle for
for (let i = 0; i < 5; i++) {
    console.log(`Itération numéro ${i}`);
}

// Boucle while
let j = 0;
while (j < 5) {
    console.log(`Itération numéro ${j}`);
    j++;
}

5. Tableaux

let fruits = ["pomme", "banane", "orange"];
console.log(fruits[0]); // Affiche "pomme"

// Méthodes utiles
fruits.push("kiwi"); // Ajoute un élément
fruits.pop(); // Retire le dernier élément

6. Fonctions

function direBonjour(nom) {
    return `Bonjour, ${nom}!`;
}
console.log(direBonjour("Alice")); // Affiche "Bonjour, Alice!"

// Fonction fléchée (ES6)
const direBonjourArrow = (nom) => `Bonjour, ${nom}!`;
console.log(direBonjourArrow("Bob")); // Affiche "Bonjour, Bob!"

7. Objets

let personne = {
    nom: "Alice",
    age: 25,
    ville: "Paris",
    direBonjour: function() {
        console.log(`Bonjour, je m'appelle ${this.nom}.`);
    }
};
console.log(personne.nom); // Affiche "Alice"
personne.direBonjour(); // Affiche "Bonjour, je m'appelle Alice."

8. Manipulation du DOM

// Sélectionner un élément
let titre = document.querySelector("h1");

// Modifier son contenu
titre.textContent = "Nouveau titre";

// Ajouter un événement
let bouton = document.querySelector("button");
bouton.addEventListener("click", () => {
    alert("Bouton cliqué !");
});

9. Bonnes pratiques

  • Utiliser const par défaut, et let si la variable doit être réaffectée.
  • Éviter d'utiliser var pour éviter les problèmes de portée.
  • Utiliser des noms de variables clairs et explicites.
  • Commenter le code complexe pour faciliter la maintenance.
