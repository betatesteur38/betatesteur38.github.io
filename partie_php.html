Résumé Complet du PHP

1. Types de données, variables et portées des variables

PHP est un langage faiblement typé. Les variables commencent par $.

<?php
    // Types de données
    $entier = 42;               // Entier
    $flottant = 3.14;           // Nombre à virgule flottante
    $chaine = "Bonjour";        // Chaîne de caractères
    $booleen = true;            // Booléen
    $nul = null;                // Null
    $tableau = array(1, 2, 3);  // Tableau

    // Portée des variables
    $globale = "Je suis globale";

    function afficherLocale() {
        global $globale; // Accès à une variable globale
        $locale = "Je suis locale";
        echo $globale; // Affiche "Je suis globale"
        // echo $locale; // Erreur en dehors de cette fonction
    }

    afficherLocale();

    // Variables statiques
    function compteur() {
        static $compte = 0;
        $compte++;
        echo $compte;
    }
    compteur(); // Affiche 1
    compteur(); // Affiche 2
?>

2. Structures de contrôle et alternatives

PHP offre plusieurs façons d'écrire des conditions et des boucles.

<?php
    // Structure if/elseif/else classique
    $heure = 14;
    if ($heure < 12) {
        echo "Bonjour!";
    } elseif ($heure < 18) {
        echo "Bon après-midi!";
    } else {
        echo "Bonsoir!";
    }

    // Structure alternative (syntaxe colon)
    if ($heure < 12):
        echo "Bonjour!";
    elseif ($heure < 18):
        echo "Bon après-midi!";
    else:
        echo "Bonsoir!";
    endif;

    // Opérateur ternaire
    $message = ($heure < 12) ? "Bonjour" : "Bonsoir";
    echo $message;

    // Switch
    $fruit = "pomme";
    switch ($fruit) {
        case "pomme":
            echo "C'est une pomme.";
            break;
        case "banane":
            echo "C'est une banane.";
            break;
        default:
            echo "Fruit inconnu.";
    }

    // Alternative pour switch
    switch ($fruit):
        case "pomme":
            echo "C'est une pomme.";
            break;
        case "banane":
            echo "C'est une banane.";
            break;
        default:
            echo "Fruit inconnu.";
    endswitch;
?>

3. Boucles

<?php
    // Boucle for
    for ($i = 0; $i < 5; $i++) {
        echo "Itération numéro $i<br>";
    }

    // Boucle for alternative
    for ($i = 0; $i < 5; $i++):
        echo "Itération numéro $i<br>";
    endfor;

    // Boucle while
    $j = 0;
    while ($j < 5) {
        echo "Itération numéro $j<br>";
        $j++;
    }

    // Boucle while alternative
    $k = 0;
    while ($k < 5):
        echo "Itération numéro $k<br>";
        $k++;
    endwhile;

    // Boucle do...while
    $l = 0;
    do {
        echo "Itération numéro $l<br>";
        $l++;
    } while ($l < 5);

    // Boucle foreach (pour les tableaux)
    $fruits = ["pomme", "banane", "orange"];
    foreach ($fruits as $fruit) {
        echo "Fruit : $fruit<br>";
    }

    // Boucle foreach alternative
    foreach ($fruits as $fruit):
        echo "Fruit : $fruit<br>";
    endforeach;
?>

4. Procédures, fonctions et paramètres

<?php
    // Fonction simple
    function direBonjour() {
        echo "Bonjour!";
    }
    direBonjour(); // Affiche "Bonjour!"

    // Fonction avec paramètres
    function direBonjourA($nom) {
        echo "Bonjour, $nom!";
    }
    direBonjourA("Alice"); // Affiche "Bonjour, Alice!"

    // Paramètres par défaut
    function direBonjourAvecDefaut($nom = "Invité") {
        echo "Bonjour, $nom!";
    }
    direBonjourAvecDefaut(); // Affiche "Bonjour, Invité!"

    // Retour de valeur
    function calculerSomme($a, $b) {
        return $a + $b;
    }
    $resultat = calculerSomme(3, 4);
    echo "La somme est $resultat"; // Affiche "La somme est 7"

    // Retour de plusieurs valeurs (via un tableau)
    function calculer($a, $b) {
        $somme = $a + $b;
        $produit = $a * $b;
        return [$somme, $produit];
    }
    list($somme, $produit) = calculer(3, 4);
    echo "Somme: $somme, Produit: $produit"; // Affiche "Somme: 7, Produit: 12"

    // Paramètres variables (variadics)
    function somme() {
        $somme = 0;
        foreach (func_get_args() as $nombre) {
            $somme += $nombre;
        }
        return $somme;
    }
    echo somme(1, 2, 3, 4); // Affiche "10"

    // Paramètres nommés (PHP 8.0+)
    function direBonjourNomme($nom, $age) {
        echo "Bonjour $nom, tu as $age ans.";
    }
    direBonjourNomme(age: 25, nom: "Alice"); // Affiche "Bonjour Alice, tu as 25 ans."
?>

5. Tableaux

<?php
    // Tableau indexé
    $couleurs = array("rouge", "vert", "bleu");
    $couleurs[] = "jaune"; // Ajoute un élément
    echo $couleurs[0]; // Affiche "rouge"

    // Tableau associatif
    $personne = [
        "nom" => "Alice",
        "âge" => 25,
        "ville" => "Paris"
    ];
    echo $personne["nom"]; // Affiche "Alice"

    // Tableau multidimensionnel
    $utilisateurs = [
        ["nom" => "Alice", "âge" => 25],
        ["nom" => "Bob", "âge" => 30]
    ];
    echo $utilisateurs[0]["nom"]; // Affiche "Alice"

    // Parcourir un tableau
    foreach ($personne as $cle => $valeur) {
        echo "$cle : $valeur<br>";
    }

    // Fonctions utiles pour les tableaux
    $nombres = [3, 1, 4, 1, 5];
    sort($nombres); // Trie le tableau
    $longueur = count($nombres); // Compte le nombre d'éléments
    $unique = array_unique($nombres); // Supprime les doublons
?>

6. Include, include_once, require, require_once

Ces instructions permettent d'inclure et d'évaluer des fichiers PHP externes.

<?php
    // include : inclut un fichier, génère un avertissement si le fichier est introuvable
    include 'fichier.php';

    // include_once : inclut un fichier une seule fois (évite les inclusions multiples)
    include_once 'fichier.php';

    // require : inclut un fichier, génère une erreur fatale si le fichier est introuvable
    require 'fichier.php';

    // require_once : inclut un fichier une seule fois, génère une erreur fatale si le fichier est introuvable
    require_once 'fichier.php';

    // Exemple d'utilisation
    // fichier.php
    <?php
        function direBonjour() {
            echo "Bonjour depuis fichier.php!";
        }
    ?>

    // index.php
    <?php
        require 'fichier.php';
        direBonjour(); // Affiche "Bonjour depuis fichier.php!"
    ?>
?>
Note : Utilisez require_once pour les fichiers essentiels (comme les bibliothèques ou les configurations) et include_once pour les fichiers optionnels.

7. Manipulation de Chaînes de Caractères

<?php
    $chaine = "Bonjour le monde";

    // Longueur d'une chaîne
    echo strlen($chaine); // Affiche 16

    // Remplacer du texte
    echo str_replace("monde", "PHP", $chaine); // Affiche "Bonjour le PHP"

    // Convertir en majuscules/minuscules
    echo strtoupper($chaine); // Affiche "BONJOUR LE MONDE"
    echo strtolower($chaine); // Affiche "bonjour le monde"

    // Extraire une sous-chaîne
    echo substr($chaine, 0, 7); // Affiche "Bonjour"
?>

9. Connexion à une Base de Données (MySQL)

<?php
    $serveur = "localhost";
    $utilisateur = "utilisateur";
    $motDePasse = "motdepasse";
    $baseDeDonnees = "ma_base";

    // Connexion
    $connexion = new mysqli($serveur, $utilisateur, $motDePasse, $baseDeDonnees);

    // Vérification de la connexion
    if ($connexion->connect_error) {
        die("La connexion a échoué: " . $connexion->connect_error);
    }

    // Exécution d'une requête
    $resultat = $connexion->query("SELECT * FROM utilisateurs");

    // Affichage des résultats
    while ($ligne = $resultat->fetch_assoc()) {
        echo "ID: " . $ligne["id"] . " - Nom: " . $ligne["nom"] . "<br>";
    }

    // Fermeture de la connexion
    $connexion->close();
?>

10. Sécurité en PHP

  • Injections SQL : Toujours utiliser des requêtes préparées.
    • $requete = $connexion->prepare("INSERT INTO utilisateurs (nom, email) VALUES (?, ?)");
$requete->bind_param("ss", $nom, $email);
$requete->execute();
  • XSS (Cross-Site Scripting) : Utiliser htmlspecialchars() pour afficher du contenu utilisateur.
  • Validation des entrées : Toujours valider et filtrer les données utilisateur avec filter_var().

11. Bonnes Pratiques

  • Utiliser des noms de variables et de fonctions explicites.

  • Utiliser des commentaires pour expliquer le code complexe.
  • Ne pas afficher les erreurs PHP en production (utiliser error_reporting(0) et configurer php.ini).
