Résumé du PHP

1. Introduction

PHP (Hypertext Preprocessor) est un langage de script côté serveur utilisé pour développer des sites web dynamiques. Il est souvent intégré dans du HTML.

2. Syntaxe de base

Les balises PHP s'ouvrent avec <?php et se ferment avec ?>.

<?php
    echo "Bonjour, monde!";
?>

3. Variables

Les variables commencent par $.

<?php
    $nom = "Alice";
    $age = 25;
    echo "Je m'appelle $nom et j'ai $age ans.";
?>

4. Structures de contrôle

Conditions

<?php
    $heure = 14;
    if ($heure < 12) {
        echo "Bonjour!";
    } elseif ($heure < 18) {
        echo "Bon après-midi!";
    } else {
        echo "Bonsoir!";
    }
?>

Boucles

<?php
    // Boucle for
    for ($i = 0; $i < 5; $i++) {
        echo "L'itération numéro $i
";
    }

    // Boucle while
    $j = 0;
    while ($j < 5) {
        echo "L'itération numéro $j
";
        $j++;
    }
?>

5. Tableaux

<?php
    $fruits = array("pomme", "banane", "orange");
    echo $fruits[0]; // Affiche "pomme"

    // Tableau associatif
    $personne = array(
        "nom" => "Alice",
        "âge" => 25,
        "ville" => "Paris"
    );
    echo $personne["nom"]; // Affiche "Alice"
?>

6. Fonctions

<?php
    function direBonjour($nom) {
        return "Bonjour, $nom!";
    }
    echo direBonjour("Alice"); // Affiche "Bonjour, Alice!"
?>

7. Connexion à une base de données (MySQL)

<?php
    $serveur = "localhost";
    $utilisateur = "utilisateur";
    $motDePasse = "motdepasse";
    $baseDeDonnees = "ma_base";

    $connexion = new mysqli($serveur, $utilisateur, $motDePasse, $baseDeDonnees);

    if ($connexion->connect_error) {
        die("La connexion a échoué: " . $connexion->connect_error);
    }
    echo "Connexion réussie!";
?>

8. Bonnes pratiques

  • Toujours valider et nettoyer les entrées utilisateur pour éviter les injections SQL.
  • Utiliser des mots de passe sécurisés pour les bases de données.
  • Éviter d'afficher les erreurs PHP en production.
  • Utiliser des commentaires pour expliquer le code complexe.
